#UXUI #Intuitive #NYC Guide

About the Project

This project involves designing a travel guide app that curates key landmarks in New York, allowing users to explore the city easily and intuitively.

The app provides photos and location details for each destination. Additionally, it includes in-depth descriptions covering the history,
unique features, and reasons to visit each site, aiming to offer users a more immersive travel experience beyond simple information delivery.
Key Features
• Map Integration: View landmark locations on the map section and navigate directly using a map app.
• Visual-Centric UI: Explore destinations intuitively through high-quality images.
• Detailed Information: Access in-depth descriptions, including history, reasons to visit, and useful tips for each location.
• Filter Function: Search for places based on specific categories of interest (e.g., culture, history, food, etc.).
